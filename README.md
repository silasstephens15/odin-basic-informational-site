This is a basic Node.js web server. I have created it in accordance with the Odin Project's Basic Informational Site project.

The index.js is just a simple Node.js server that takes in a url and returns the appropriate html. It accounts for 404 errors. The html pages do not contain any styling or much content, see my other projects for examples of my front end work.

The app.js uses Express framework to accomplish the same thing.
